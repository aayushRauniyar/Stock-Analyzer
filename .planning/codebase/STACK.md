# Tech Stack

## Programming Languages
- **Python**: Primary backend language (3.14.3 active).
- **JavaScript/JSX**: Primary frontend language.

## Runtime & Frameworks
- **Frontend**: React 19, Vite (Vanilla CSS for styling).
- **Backend**: Python HTTP Server/Flask API (`server.py`).

## Core Dependencies
- `yfinance` >= 0.2.38 (Market data)
- `pandas`, `numpy`, `ta` (Technical indicators)
- `alpaca-trade-api` >= 3.0.2 (Trading execution)
- `groq` >= 0.11.0 (AI LLM inference)
- `mcp` (Model Context Protocol server)
- `schedule`, `pytz` (Task scheduling)

## Build & Deployment
- Node package manager (NPM/NPX) for JS.
- pip/uv for Python dependencies
- Runs locally using `npm run dev` and `python server.py`.
